Información para clientes

Política de privacidad

Última actualización: 27 de junio de 2026.

Mommy Crafts es un emprendimiento independiente de productos personalizados. Utilizamos los datos entregados por cada cliente únicamente para gestionar cuentas, pedidos, personalizaciones, pagos, entregas y atención posterior.

1. Datos que podemos solicitar

  • Nombre y correo electrónico.
  • Teléfono para coordinar pedidos y entregas.
  • Dirección y comuna cuando se elige despacho.
  • RUT, solamente cuando el cliente decide proporcionarlo o resulta necesario para el pedido.
  • Contenido, imágenes y textos enviados para personalizar un producto.
  • Información técnica básica de seguridad, como registros de acceso y solicitudes al servidor.

2. Para qué usamos la información

  • Crear y administrar cuentas de clientes.
  • Preparar, confirmar, cobrar y entregar pedidos.
  • Elaborar productos según las instrucciones de personalización.
  • Responder consultas, reclamos o solicitudes de actualización.
  • Proteger el sitio frente a accesos no autorizados y abuso automatizado.
  • Cumplir obligaciones operativas, contables o legales aplicables.

3. Servicios tecnológicos utilizados

El sitio puede utilizar Netlify para el frontend, Render para la API, MongoDB Atlas para la base de datos y Cloudinary para imágenes. Cuando Mercado Pago sea habilitado, el pago se procesará en su plataforma. Mommy Crafts no almacena números completos de tarjetas ni claves bancarias.

4. Conservación y seguridad

Conservamos la información mientras sea necesaria para gestionar la cuenta, completar pedidos, atender solicitudes y mantener los respaldos u obligaciones que correspondan. Aplicamos acceso restringido, contraseñas protegidas mediante hash, conexiones HTTPS, límites de solicitudes y registros de seguridad.

5. Solicitudes sobre tus datos

Puedes pedir acceso, corrección o eliminación de la información asociada a tu cuenta. Algunas partes de un pedido podrían conservarse durante el período necesario para respaldar la compra o cumplir obligaciones aplicables.

6. Contacto

Para consultas de privacidad, utiliza el WhatsApp oficial de Mommy Crafts .

Esta política describe el funcionamiento actual del emprendimiento y podrá actualizarse cuando cambien sus procesos o servicios.
