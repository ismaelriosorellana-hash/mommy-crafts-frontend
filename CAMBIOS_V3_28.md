# Mommy Crafts Frontend V3.28

## Inicio y banners

- El botón de cada banner puede dirigir a una categoría específica, una sección o una URL personalizada.
- Se agregaron indicadores circulares para elegir manualmente el banner.
- El cambio de banner usa una transición suave y mantiene la rotación automática.

## Productos y personalización

- Se agregaron tallas en tarjetas de catálogo, detalle de producto y personalización avanzada.
- Las tallas se administran desde el panel de productos.
- La talla aparece en carrito, checkout, WhatsApp, cuenta del cliente y detalle administrativo del pedido.
- Elegir un color ya no avanza automáticamente al siguiente paso.
- El precio avanzado usa el precio actual del producto o variante.
- La variante blanca se prioriza como imagen inicial cuando está disponible.
- La imagen subida puede seleccionarse y moverse tanto en el paso 5 como en el paso 6.
- Descripción, preparación mínima, envío y retiro aparecen inicialmente colapsados.

## Checkout y pedidos

- El calendario se abre al hacer clic y solo está disponible para retiro.
- Se incorporó selección de zona de envío:
  - Provincia de Santiago: $4.000, entre 5 y 7 días hábiles.
  - Otros sectores: Chilexpress por pagar, desde 5 días hábiles.
- El resumen del checkout actualiza subtotal, envío y total estimado.
- Las instrucciones muestran el nombre del producto en el título.
- Se corrigió el tamaño de las opciones de pago en escritorio.
- Se eliminó el parpadeo del botón para enviar comprobante.
- El botón de WhatsApp quedó centrado y con mayor separación.
- En administración, Entrega, Transferencia bancaria, Fecha, Productos y Actualizar pedido son secciones colapsables.

## Validaciones

- 27 archivos HTML revisados sin identificadores duplicados ni referencias locales faltantes.
- Todos los archivos JavaScript del frontend pasan validación sintáctica.
