# Frontend V3.32 — Boutique lúdica

## Rediseño público

- Nueva identidad visual original inspirada en tiendas infantiles modernas, sin copiar código ni recursos de terceros.
- Paleta coral, lavanda, menta, amarillo y azul suave.
- Tipografías redondeadas, mayor jerarquía visual y espaciado más limpio.
- Encabezado flotante, navegación en píldoras, buscador y carrito destacados.
- Banner principal inmersivo con controles, transiciones y formas decorativas.
- Franja de confianza y mosaico promocional en la página principal.
- Categorías arqueadas, tarjetas de producto suaves y microanimaciones.
- Rediseño de catálogo, detalle de producto, carrito, checkout, modales, cuenta y páginas informativas.
- Responsive para escritorio, tablet y teléfono.
- Respeta `prefers-reduced-motion` para accesibilidad.

## Compatibilidad

- Conserva todos los identificadores, formularios y selectores utilizados por JavaScript.
- Conserva catálogo dinámico, personalización simple y avanzada, carrito, envío gratis, checkout, Mercado Pago, pedidos, cuenta de cliente y CMS visual.
- Sigue conectado al backend ya configurado; no requiere cambios de API ni nuevas variables de entorno.
- Los colores, logo y cinta superior existentes continúan administrándose desde Apariencia.
- El contenido y orden de las secciones compatibles continúan administrándose desde Editor del sitio.
