MOMMY CRAFTS FRONTEND V3.8
==========================

Cambios:

1. Selector de color en la página del producto.
2. Cada color muestra las imágenes específicas de ese producto.
3. La galería completa cambia al seleccionar un color.
4. El color seleccionado se guarda en el carrito.
5. La imagen del carrito corresponde al color elegido.
6. Stock, precio, precio anterior y SKU pueden configurarse por color.
7. Las tarjetas muestran los colores disponibles.
8. Cuando un producto tiene colores, el botón de la tarjeta dice
   "Elegir color" y lleva al detalle.
9. El paso Color del modal también utiliza las imágenes y códigos
   de color definidos en MongoDB.

Después de reemplazar:
- Mantén el backend encendido.
- Abre el frontend con Live Server.
- Presiona Ctrl + F5.
