# Mommy Crafts Frontend V3.31

## CMS visual propio

- Nueva sección administrativa **Editor del sitio**.
- Edición de páginas, SEO, visibilidad, anchos, fondos y separación de bloques.
- Reordenamiento y ocultamiento de secciones principales.
- Creación de páginas nuevas sin agregar archivos HTML adicionales.
- Bloques administrables: banner, texto, imagen y texto, CTA, aviso, productos, categorías, espaciadores y separadores.
- Menú principal y pie de página editables.
- Diseño global de botones, tarjetas, ventanas, tipografía y encabezado.
- Apariencia y orden del menú del panel administrador editables.
- Nueva página genérica `pagina.html` para contenidos creados desde administración.
- Runtime público `site-studio.js` y estilos `site-studio.css` aplicados a toda la tienda.

## Límite de seguridad

El editor no permite modificar código JavaScript, credenciales, pagos, autenticación, validaciones ni lógica de pedidos. Crear una nueva función administrativa sigue requiriendo desarrollo seguro del backend.
